# Super Store Sales & Profit Analysis Dashboard

## 📊 Project Overview
This project involves a comprehensive data analysis of a **Retail Super Store** using Power BI. The goal is to track key business metrics, visualize sales trends, and provide a 15-day sales forecast to help stakeholders make data-driven decisions regarding inventory and marketing strategies.

---

## 🚀 Key Features & Insights
- **KPI Tracking:** Real-time visibility into Total Sales ($2M), Profit ($286K), Quantity Sold (38K), and average Delivery Days.
- **Interactive Filtering:** Users can slice data by **Region** (Central, East, South, West) and **City** for granular insights.
- **Sales Forecasting:** Built a 15-day time-series forecast using Power BI's built-in analytics to predict upcoming revenue.
- **Product Analysis:** identified **Phones** and **Chairs** as top revenue-generating sub-categories.
- **Regional Performance:** The **West** region contributes the highest sales (32%), while the **Central** region shows growth opportunities.
- **Profitability Deep-Dive:** Analyzed profit margins across different Categories (Technology leads in profit).

---

## 🛠️ Tech Stack
- **Data Visualization:** Power BI Desktop
- **Data Cleaning:** Power Query Editor (ETL Process)
- **Calculations:** DAX (Data Analysis Expressions) for custom measures and KPIs.
- **Forecasting:** Time Series Analysis.

1. Clone this repository or download the `.pbix` file.
2. Open the file in **Power BI Desktop**.
3. Use the slicers on the top and sides of the dashboard to interact with the data.
